# LifeHackQuiz App

Overview

LifeHackQuiz is a native Android application developed using Kotlin. The app presents users with statements about popular “life hacks” and challenges them to decide whether they are real or just urban myths.

The goal is to help users think critically about information they encounter online while learning useful and safe tips.

Features

* Simple and clean user interface
* Interactive quiz format
* True or False answer selection
* Instant feedback on answers
* Score tracking system


How It Works

1. Launch the app
2. Tap the **Start** button
3. Read each life hack statement
4. Choose whether it is **True** or **False**
5. Receive immediate feedback
6. View your final score at the end

Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* Android SDK


Project Structure

* `MainActivity.kt` – Handles the welcome screen
* `QuizActivity.kt` – Manages quiz logic and questions
* `ResultActivity.kt` – Displays final score
* `res/layout` – Contains XML UI designs



 Installation

1. Clone the repository:

   bash
   git clone https://gitlab.com/your-username/LifeHackQuiz.git
2. Open the project in Android Studio
3. Build and run the app on an emulator or Android device
